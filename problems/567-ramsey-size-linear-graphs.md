# Erdős Problem #567

## Title
Ramsey size linear graphs

## Status
Open

## Source
https://www.erdosproblems.com/567

## Problem statement

Let $G$ be either $Q_3$ or $K_{3,3}$ or $H_5$ (the last formed by adding two vertex-disjoint chords to $C_5$). Is it true that, if $H$ has $m$ edges and no isolated vertices, then
\[
R(G,H)\ll m?
\]

## Visual idea

The problem asks whether certain fixed graphs \(G\) are Ramsey size linear.

We fix one of three graphs:

\[
Q_3,\qquad K_{3,3},\qquad H_5.
\]

Then for every graph \(H\) with \(m\) edges and no isolated vertices, we ask whether the Ramsey number \(R(G,H)\) grows only linearly in \(m\).

The poster should visually emphasize:
- the three candidate graphs \(Q_3\), \(K_{3,3}\), and \(H_5\),
- a variable graph \(H\) with \(m\) edges,
- the Ramsey expression \(R(G,H)\),
- and the desired linear bound \(R(G,H)\ll m\).
