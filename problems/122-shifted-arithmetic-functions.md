# Erdős Problem #122

## Title
Shifted arithmetic functions

## Status
Open

## Source
https://www.erdosproblems.com/122

## Problem statement

For which number theoretic functions $f$ is it true that, for any $F(n)$ such that $F(n)/f(n)\to 0$ for almost all $n$, there are infinitely many $x$ such that
\[
\frac{\#\{ n\in \mathbb{N} : n+f(n)\in (x,x+F(x))\}}{F(x)}\to \infty?
\]

## Visual idea

The problem studies shifted values of the form

\[
n+f(n).
\]

For a short interval \((x,x+F(x))\), count how many shifted values \(n+f(n)\) land inside it.

The question asks whether, for certain arithmetic functions \(f\), these shifted values must cluster extremely densely inside infinitely many short intervals.

The poster should visually emphasize:
- a number line,
- shifted points \(n\mapsto n+f(n)\),
- a short interval window \((x,x+F(x))\),
- many shifted values landing inside the same narrow window,
- and the density ratio becoming very large.
