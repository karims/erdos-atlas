# Erdős Problem #428

## Title
Prime translates

## Status
Open

## Source
https://www.erdosproblems.com/428

## Problem statement

Is there a set $A\subseteq \mathbb{N}$ such that, for infinitely many $n$, all of $n-a$ are prime for all $a\in A$ with $0<a<n$ and
\[
\liminf\frac{\lvert A\cap [1,x]\rvert}{\pi(x)}>0?
\]

## Visual idea

Choose a set \(A\subseteq \mathbb{N}\) that is not too sparse compared with the primes.

For infinitely many \(n\), reflect or translate the visible part of \(A\) below \(n\):

\[
a \mapsto n-a.
\]

The condition asks that every such translated value \(n-a\) is prime.

The poster should visually emphasize:
- a set \(A\) inside the natural numbers,
- a special integer \(n\),
- arrows from \(a\in A\) to \(n-a\),
- the translated values landing on primes,
- and the density condition comparing \(A\) with \(\pi(x)\).
