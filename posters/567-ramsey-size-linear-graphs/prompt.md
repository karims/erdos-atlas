# Poster prompt: Erdős Problem #567

Create a vertical mathematical poster in the Erdős Atlas visual style:
ivory parchment background, black serif typography, deep red accents, delicate divider lines, elegant framing, and a refined mathematical-art composition.

Poster title:
Ramsey Size Linear?

Subtitle:
Erdős Problem #567

Small label:
When does \(R(G,H)\) grow linearly with \(e(H)\)?

Use the exact problem statement in a compact problem panel:

Let $G$ be either $Q_3$ or $K_{3,3}$ or $H_5$ (the last formed by adding two vertex-disjoint chords to $C_5$). Is it true that, if $H$ has $m$ edges and no isolated vertices, then
\[
R(G,H)\ll m?
\]

Main visual structure:
1. A panel titled “The Problem”.
2. A panel titled “The fixed graph \(G\)” showing three small graph diagrams:
   - \(Q_3\), the cube graph,
   - \(K_{3,3}\), a bipartite 3-by-3 graph,
   - \(H_5\), a 5-cycle with two vertex-disjoint chords.
3. A panel titled “The variable graph \(H\)”:
   - draw an abstract graph \(H\),
   - label it \(m\) edges,
   - note “no isolated vertices.”
4. A central formula panel:
   \[
   R(G,H)\ll m
   \]
5. A Ramsey interpretation panel:
   - in every sufficiently large red/blue edge-colouring,
   - one colour contains \(G\),
   - or the other colour contains \(H\).
6. A question panel:
   \[
   \text{Are } Q_3,\ K_{3,3},\ H_5 \text{ Ramsey size linear?}
   \]
7. Optional context note:
   \(H_5\) is also \(K_4^*\), obtained from \(K_4\) by subdividing one edge.
