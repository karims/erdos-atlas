# Source notes

## Problem ID
509

## Source link
https://www.erdosproblems.com/509

## Exact statement

Let $f(z)\in\mathbb{C}[z]$ be a monic non-constant polynomial. Can the set
\[
\{ z\in \mathbb{C} : \lvert f(z)\rvert \leq 1\}
\]
be covered by a set of circles the sum of whose radii is $\leq 2$?

## Short label
Polynomial lemniscate covering

## Status
Open

## Tags
analysis, polynomials

## Notes

The page states that Cartan proved this is true with \(2\) replaced by

\[
2e.
\]

Pommerenke improved this to

\[
2.59.
\]

Pommerenke also proved that the constant \(2\) is achievable if the set is connected.

The generalisation to higher dimensions was asked by Erdős as Problem 4.23 in [Ha74].

## Visual metaphor

A polynomial carves out a shadow in the complex plane:
\[
|f(z)|\leq 1.
\]
The question is whether that shadow can always be hidden beneath circles with total radius no more than \(2\).
