# Poster prompt: Erdős Problem #509

Create a vertical mathematical poster in the Erdős Atlas visual style:
ivory parchment background, black serif typography, deep red accents, delicate divider lines, elegant framing, and a refined mathematical-art composition.

Poster title:
Covering Polynomial Shadows

Subtitle:
Erdős Problem #509

Small label:
Covering polynomial lemniscates by circles

Use the exact problem statement in a compact problem panel:

Let $f(z)\in\mathbb{C}[z]$ be a monic non-constant polynomial. Can the set
\[
\{ z\in \mathbb{C} : \lvert f(z)\rvert \leq 1\}
\]
be covered by a set of circles the sum of whose radii is $\leq 2$?

Main visual structure:
1. A panel titled “The Problem”.
2. A panel titled “The Lemniscate” explaining:
   - \(f(z)\) is monic and non-constant,
   - the region is \(L_f=\{z\in\mathbb C:|f(z)|\leq 1\}\),
   - this is a polynomial sublevel set in the complex plane.
3. A central visual:
   - draw a complex plane,
   - show a lemniscate-shaped region with one or several components,
   - overlay circles covering the entire region,
   - label the circle radii \(r_1,r_2,r_3,\ldots\).
4. A radius-budget panel:
   \[
   \sum_i r_i\leq 2.
   \]
5. A known constants panel:
   \[
   2e \quad\longrightarrow\quad 2.59 \quad\longrightarrow\quad 2?
   \]
6. A connected-case note:
   \[
   \text{If }L_f\text{ is connected, }2\text{ is achievable.}
   \]
7. A mystery panel:
   \[
   \text{Does total radius }2\text{ always suffice?}
   \]

Design rules:
