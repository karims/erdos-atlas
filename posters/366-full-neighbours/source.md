## Problem ID
366

## Source link
https://www.erdosproblems.com/366

## Exact statement

Are there any $2$-full $n$ such that $n+1$ is $3$-full? That is, if $p\mid n$ then $p^2\mid n$ and if $p\mid n+1$ then $p^3\mid n+1$.

## Short label
Full neighbours

## Status
Open (verifiable)

## Tags
number theory, powerful numbers

## Notes

The site remarks that:
- $8$ is $3$-full and $9$ is $2$-full.
- $12167 = 23^3$ and $12168 = 2^3 3^2 13^2$ is another example.
- No other examples are known for $n < 10^{22}$.

## Visual metaphor
Two neighboring integers, each built from prime-power blocks:
one “square-deep”, the other “cube-deep”.
