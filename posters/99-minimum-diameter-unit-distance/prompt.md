# Poster prompt: Erdős Problem #99

Create a vertical mathematical poster in the Erdős Atlas visual style:
ivory parchment background, black serif typography, deep red accents, delicate divider lines, elegant framing, and a refined mathematical-art composition.

Poster title:
The Smallest Diameter Trap

Subtitle:
Erdős Problem #99

Small label:
Minimum-distance point sets and unit equilateral triangles

Use the exact problem statement in a compact problem panel:

Let $A\subseteq\mathbb{R}^2$ be a set of $n$ points with minimum distance equal to 1, chosen to minimise the diameter of $A$. If $n$ is sufficiently large then must there be three points in $A$ which form an equilateral triangle of size 1?

Main visual structure:
1. A panel titled “The Problem”.
2. A panel titled “The Constraints” explaining:
   - \(A\subseteq\mathbb{R}^2\) has \(n\) points,
   - minimum distance is \(1\),
   - among all such sets, choose one with minimum diameter.
3. A central visual:
   - draw a compact set of points inside a smallest enclosing circle,
   - show the diameter as a red chord across the circle,
   - show a faint triangular lattice in the background,
   - highlight three points forming a unit equilateral triangle.
4. A minimum-distance panel:
   \[
   \min_{a\neq b}|a-b|=1.
   \]
5. A diameter panel:
   \[
   \operatorname{diam}(A)=\max_{a,b\in A}|a-b|.
   \]
6. A triangle question panel:
   \[
   \triangle\text{ of side }1\quad\text{must appear?}
   \]
7. A context panel:
   - triangular lattice is asymptotically optimal,
   - false for \(n=4\) using a square.
