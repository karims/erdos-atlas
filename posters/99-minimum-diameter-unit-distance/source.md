# Source notes

## Problem ID
99

## Source link
https://www.erdosproblems.com/99

## Exact statement

Let $A\subseteq\mathbb{R}^2$ be a set of $n$ points with minimum distance equal to 1, chosen to minimise the diameter of $A$. If $n$ is sufficiently large then must there be three points in $A$ which form an equilateral triangle of size 1?

## Short label
Minimum diameter unit-distance sets

## Status
Open

## Prize
$100

## Tags
geometry, distances

## Notes

The page notes that Thue proved the minimal such diameter is achieved asymptotically by points in a triangular lattice intersected with a circle.

Erdős believed such a set must have very large intersection with the triangular lattice, perhaps as many as

\[
(1-o(1))n
\]

points.

Erdős wrote that he could not prove the conjecture and was surprised that B. H. Sendov and M. Simonovits doubted its truth.

The stated problem is false for \(n=4\), for example by taking the vertices of a square.

## Visual metaphor

A finite point set is packed as tightly as possible while keeping every pair at distance at least \(1\).  
Does this forced compactness eventually create a unit equilateral triangle?
