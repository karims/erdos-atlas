# Poster prompt: Erdős Problem #97

Create a vertical mathematical poster in the Erdős Atlas visual style:
ivory parchment background, black serif typography, deep red accents, delicate divider lines, elegant framing, and a refined mathematical-art composition.

Poster title:
Four at the Same Distance?

Subtitle:
Erdős Problem #97

Small label:
Equal-distance vertices in convex polygons

Use the exact problem statement in a compact problem panel:

Does every convex polygon have a vertex with no other $4$ vertices equidistant from it?

Main visual structure:
1. A panel titled “The Problem”.
2. A panel titled “What is being tested?” explaining:
   - choose a vertex \(v\) of a convex polygon,
   - measure distances from \(v\) to all other vertices,
   - ask whether some vertex has no distance repeated four times.
3. A central visual:
   - draw a convex polygon with many vertices,
   - highlight one vertex \(v\) in deep red,
   - draw circles centered at \(v\),
   - show one circle passing through four other vertices as the forbidden pattern,
   - contrast with a “good” vertex where no circle contains four other vertices.
4. A formula panel:
   \[
   |v-w_1|=|v-w_2|=|v-w_3|=|v-w_4|
   \]
   is the pattern to avoid.
5. A question panel:
   \[
   \text{Must some vertex avoid this?}
   \]
6. A context panel:
   - the analogous \(3\)-vertex version is false,
   - Danzer gave a \(9\)-point example,
   - Fishburn and Reeds gave a \(20\)-point example.
7. A connection panel:
   If true for \(k+1\), it gives a \(kn\) bound for unit distances in convex polygons.

Design rules:
- Keep the house style close to the initial Erdős Atlas posters.
