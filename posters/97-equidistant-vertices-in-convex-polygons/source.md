# Source notes

## Problem ID
97

## Source link
https://www.erdosproblems.com/97

## Exact statement

Does every convex polygon have a vertex with no other $4$ vertices equidistant from it?

## Short label
Equidistant vertices in convex polygons

## Status
Open (falsifiable)

## Prize
$100

## Tags
geometry, distances, convex

## Notes

The page states that Erdős originally conjectured this with no \(3\) vertices equidistant, but Danzer found a convex polygon on \(9\) points such that every vertex has three vertices equidistant from it.

Fishburn and Reeds found a convex polygon on \(20\) points such that every vertex has three vertices equidistant from it, with the same distance for all vertices.

The page asks whether the statement becomes true for \(4\) equidistant vertices.

It also notes that if the problem holds for \(k+1\) vertices, then by induction it implies an upper bound of \(kn\) for Erdős Problem #96.

## Visual metaphor

A vertex sends out distance circles across a convex polygon.  
The problem asks whether some vertex must avoid having four companions on the same circle.
