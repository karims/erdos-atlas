# Poster prompt: Erdős Problem #122

Create a vertical mathematical poster in the Erdős Atlas visual style:
ivory parchment background, black serif typography, deep red accents, delicate divider lines, elegant framing, and a refined mathematical-art composition.

Poster title:
Crowded Shifts

Subtitle:
Erdős Problem #122

Small label:
Clustering of shifted arithmetic functions

Use the exact problem statement in a compact problem panel:

For which number theoretic functions $f$ is it true that, for any $F(n)$ such that $F(n)/f(n)\to 0$ for almost all $n$, there are infinitely many $x$ such that
\[
\frac{\#\{ n\in \mathbb{N} : n+f(n)\in (x,x+F(x))\}}{F(x)}\to \infty?
\]

Main visual structure:
1. A panel titled “The Problem”.
2. A panel titled “What is being shifted?” explaining:
   - choose a number-theoretic function \(f\),
   - each integer \(n\) moves to \(n+f(n)\),
   - examine where these shifted values land.
3. A central visual:
   - draw a horizontal number line,
   - show several original points \(n\),
   - draw arrows from \(n\) to \(n+f(n)\),
   - show many arrows landing inside a narrow highlighted interval \((x,x+F(x))\).
4. A short-window panel:
   \[
   (x,x+F(x))
   \]
   with the condition
   \[
   F(n)/f(n)\to 0
   \]
   for almost all \(n\).
5. A density-ratio panel:
   \[
   \frac{\#\{n:n+f(n)\in(x,x+F(x))\}}{F(x)}
   \to\infty.
   \]
6. A context panel:
   - works for \(\tau(n)\) and \(\omega(n)\),
   - probably fails for \(\phi(n)\) and \(\sigma(n)\).
