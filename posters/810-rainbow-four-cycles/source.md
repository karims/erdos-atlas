# Source notes

## Problem ID
810

## Source link
https://www.erdosproblems.com/810

## Exact statement

Does there exist some $\epsilon>0$ such that, for all sufficiently large $n$, there exists a graph $G$ on $n$ vertices with at least $\epsilon n^2$ many edges such that the edges can be coloured with $n$ colours so that every $C_4$ receives $4$ distinct colours?

## Short label
Rainbow four-cycles

## Status
Open

## Tags
graph theory, Ramsey theory

## Notes

This is a problem of Burr, Erdős, Graham, and Sós.

The page remarks that they believed the answer is no.

Equivalently, the problem can be phrased using an anti-Ramsey number:
\[
\chi_S(n,\epsilon n^2,C_4)\leq n
\]
for all sufficiently large \(n\).

## Visual metaphor

A dense graph is woven with only \(n\) edge colours, but every square-shaped cycle must display four different colours.
