# Poster prompt: Erdős Problem #165

Create a vertical mathematical poster in the Erdős Atlas visual style:
ivory parchment background, black serif typography, deep red accents, delicate divider lines, elegant framing, and a refined mathematical-art composition.

Poster title:
The Triangle–Independence Threshold

Subtitle:
Erdős Problem #165

Small label:
The Ramsey number \(R(3,k)\)

Use the exact problem statement in a compact problem panel:

Give an asymptotic formula for $R(3,k)$.

Add a short explanatory panel:

\(R(3,k)\) is the smallest \(N\) such that every graph on \(N\) vertices contains either a triangle \(K_3\) or an independent set of size \(k\).

Main visual structure:
1. A panel titled “The Problem”.
2. A panel titled “What does \(R(3,k)\) mean?” explaining:
   - every graph on \(N\) vertices must contain either \(K_3\),
   - or an independent set of size \(k\).
3. A central visual:
   - a large graph drawn as a network of black vertices and thin edges,
   - a highlighted triangle \(K_3\) on one side,
   - a highlighted independent set of \(k\) isolated vertices on the other side,
   - the graph balanced between these two forbidden structures.
4. A bounds panel:
   \[
   (c+o(1))\frac{k^2}{\log k}
   \leq
   R(3,k)
   \leq
   (1+o(1))\frac{k^2}{\log k}.
   \]
5. A mystery panel:
   \[
   R(3,k)\stackrel{?}{\sim} C\frac{k^2}{\log k}
   \]
   with the question: what is the correct constant \(C\)?
6. Include a small prize badge:
   $250 prize

Design rules:
- Keep the house style close to the initial Erdős Atlas posters.
- Use ivory, black, and deep red.
- Avoid overloading the poster with historical details.
- Make the central idea visual: every large enough graph is forced to contain one of two structures.
- Keep the triangle and independent set visually clear.
