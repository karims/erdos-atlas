# Poster prompt: Erdős Problem #968

Create a vertical mathematical poster in the Erdős Atlas visual style:
ivory parchment background, black serif typography, deep red accents, delicate divider lines, elegant framing, and a refined mathematical-art composition.

Poster title:
Rising Prime Ratios

Subtitle:
Erdős Problem #968

Small label:
When does \(p_n/n\) increase?

Use the exact problem statement in a compact problem panel:

Let $u_n=p_n/n$, where $p_n$ is the $n$th prime. Does the set of $n$ such that $u_n<u_{n+1}$ have positive density?

Main visual structure:
1. A panel titled “The Problem”.
2. A panel titled “The ratio sequence” explaining:
   - \(p_n\) is the \(n\)th prime,
   - \(u_n=p_n/n\),
   - ask when \(u_n<u_{n+1}\).
3. A central visual:
   - draw a discrete curve or staircase of values \(u_n\),
   - highlight upward steps in deep red,
   - mark the condition \(u_n<u_{n+1}\).
4. A formula panel:
   \[
   \frac{p_n}{n}<\frac{p_{n+1}}{n+1}
   \]
5. A prime-gap interpretation panel:
   \[
   u_n<u_{n+1}
   \iff
   p_{n+1}-p_n>\frac{p_n}{n}
   \approx \log n.
   \]
6. A density question panel:
   \[
   \#\{m\leq x:u_m<u_{m+1}\}\geq cx?
   \]
   for some \(c>0\) and all large \(x\)?
7. Optional contrast note:
   Erdős and Prachar proved the decreasing set \(u_n>u_{n+1}\) has positive density.
