# Poster prompt: Erdős Problem #901

Create a vertical mathematical poster in the Erdős Atlas visual style:
ivory parchment background, black serif typography, deep red accents, delicate divider lines, elegant framing, and a refined mathematical-art composition.

Poster title:
Forcing a Third Colour

Subtitle:
Erdős Problem #901

Small label:
Minimal non-2-colourable uniform hypergraphs

Use the exact problem statement in a compact problem panel:

Let $m(n)$ be minimal such that there is an $n$-uniform hypergraph with $m(n)$ edges which is $3$-chromatic. Estimate $m(n)$.

Main visual structure:
1. A panel titled “The Problem”.
2. A panel titled “What is \(m(n)\)?” explaining:
   - every edge has exactly \(n\) vertices,
   - the hypergraph is \(3\)-chromatic,
   - \(m(n)\) is the minimum number of edges needed.
3. A central visual:
   - draw a set of vertices as small points,
   - draw several \(n\)-element hyperedges as translucent loops or rounded regions,
   - use red and black vertex colouring,
   - show one hyperedge forced to be monochromatic.
4. A Property B panel:
   - 2-colourable means every edge meets both colours,
   - failing Property B means some monochromatic edge appears in every 2-colouring.
5. A known values panel:
   \[
   m(2)=3,\qquad m(3)=7,\qquad m(4)=23.
   \]
6. A bounds panel:
   \[
   2^n \ll m(n) \ll n^2 2^n.
   \]
7. A conjectural scale panel:
   \[
   m(n)\stackrel{?}{\asymp} n2^n.
   \]

Design rules:
- Keep the house style close to the initial Erdős Atlas posters.
- Use ivory, black, and deep red.
- Do not make the poster too dense.
- Make the central visual about hyperedges as sets, not ordinary graph edges.
- Keep \(m(n)\), \(3\)-chromatic, and \(n2^n\) visually prominent.
