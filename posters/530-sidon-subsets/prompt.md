# Poster prompt: Erdős Problem #530

Create a vertical mathematical poster in the Erdős Atlas visual style:
ivory parchment background, black serif typography, deep red accents, delicate divider lines, elegant framing, and a refined mathematical-art composition.

Poster title:
Sidon Subsets

Subtitle:
Erdős Problem #530

Small label:
Pair-sum uniqueness inside finite sets

Use the exact problem statement in a compact problem panel:

Let $\ell(N)$ be maximal such that in any finite set $A\subset \mathbb{R}$ of size $N$ there exists a Sidon subset $S$ of size $\ell(N)$ (i.e. the only solutions to $a+b=c+d$ in $S$ are the trivial ones). Determine the order of $\ell(N)$.

In particular, is it true that $\ell(N)\sim N^{1/2}$?

Main visual structure:
1. A panel titled “The Problem”.
2. A panel titled “What is a Sidon subset?” explaining:
   - choose $S\subset A$
   - pair sums $a+b$ are unique except for trivial reorderings
   - collisions $a+b=c+d$ are forbidden unless the pairs are the same
3. A central visual:
   - a large set $A$ drawn as many black and faint red points on a line or constellation
   - a selected subset $S$ highlighted in deep red
   - pair-sum arcs from selected elements mapping to distinct sum marks
4. A formula panel:
   \[
   a+b=c+d
   \]
   allowed only in trivial cases.
5. A growth question panel emphasizing:
   \[
   |S|=\ell(N)
   \]
   and
   \[
   \ell(N)\stackrel{?}{\sim}N^{1/2}.
   \]
6. Optional known-bounds panel:
   \[
   N^{1/2}\ll \ell(N)\leq (1+o(1))N^{1/2}
   \]
   with a note that the correct constant is unknown.
