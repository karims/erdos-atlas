# Poster prompt: Erdős Problem #428

Create a vertical mathematical poster in the Erdős Atlas visual style:
ivory parchment background, black serif typography, deep red accents, delicate divider lines, elegant framing, and a refined mathematical-art composition.

Poster title:
Prime Translates

Subtitle:
Erdős Problem #428

Small label:
A dense set whose reflected translates land on primes

Use the exact problem statement in a compact problem panel:

Is there a set $A\subseteq \mathbb{N}$ such that, for infinitely many $n$, all of $n-a$ are prime for all $a\in A$ with $0<a<n$ and
\[
\liminf\frac{\lvert A\cap [1,x]\rvert}{\pi(x)}>0?
\]

Main visual structure:
1. A panel titled “The Problem”.
2. A panel titled “What is being asked?” explaining:
   - choose a set \(A\subseteq \mathbb{N}\),
   - for infinitely many \(n\), every \(a\in A\) with \(0<a<n\) must satisfy \(n-a\) is prime,
   - \(A\) should have positive lower density relative to the primes.
3. A central visual:
   - draw a horizontal number line,
   - mark a special point \(n\),
   - show selected red points \(a\in A\) to the left of \(n\),
   - draw reflected arrows from each \(a\) through \(n\) or toward labels \(n-a\),
   - mark each \(n-a\) as prime using small black/red prime tokens.
4. A formula panel:
   \[
   a\in A,\ 0<a<n \Rightarrow n-a\in\mathbb{P}
   \]
5. A density panel:
   \[
   \liminf_{x\to\infty}
   \frac{|A\cap[1,x]|}{\pi(x)}>0
   \]
6. A mystery panel:
   \[
   \text{Can this happen for infinitely many }n?
   \]

Design rules:
- Keep the house style close to the initial Erdős Atlas posters.
- Use ivory, black, and deep red.
- Do not overcrowd the poster.
- Make the central visual about translation/reflection: \(a\mapsto n-a\).
- Make the density condition visually secondary but clearly present.
- Keep all mathematical text crisp and legible.
