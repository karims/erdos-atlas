## Problem ID
428

## Source link
https://www.erdosproblems.com/428

## Exact statement

Is there a set $A\subseteq \mathbb{N}$ such that, for infinitely many $n$, all of $n-a$ are prime for all $a\in A$ with $0<a<n$ and
\[
\liminf\frac{\lvert A\cap [1,x]\rvert}{\pi(x)}>0?
\]

## Short label
Prime translates

## Status
Open

## Tags
number theory, primes

## Notes

The page notes that Erdős and Graham could show this is true, assuming the prime \(k\)-tuple conjecture, if \(\liminf\) is replaced by \(\limsup\).

## Visual metaphor

A set \(A\) casts reflected shadows through a moving point \(n\).  
For infinitely many \(n\), every visible shadow \(n-a\) must land on a prime.
