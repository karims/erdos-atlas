# Source notes

## Problem ID
234

## Source link
https://www.erdosproblems.com/234

## Exact statement

For every $c\geq 0$ the density $f(c)$ of integers for which
\[
\frac{p_{n+1}-p_n}{\log n}< c
\]
exists and is a continuous function of $c$.

## Short label
Normalized prime gaps

## Status
Open

## Tags
number theory, primes

## Visual metaphor
Prime gaps fluctuate, but after normalization one asks whether they approach a stable distribution described by the function $f(c)$.
