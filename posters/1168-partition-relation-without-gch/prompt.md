# Poster prompt: Erdős Problem #1168

Create a vertical mathematical poster in the Erdős Atlas visual style:
ivory parchment background, black serif typography, deep red accents, delicate divider lines, elegant framing, and a refined mathematical-art composition.

Poster title:
A Broken Infinite Arrow

Subtitle:
Erdős Problem #1168

Small label:
Partition relations beyond the continuum hypothesis

Use the exact problem statement in a compact problem panel:

Prove that
\[
\aleph_{\omega+1}\not\to (\aleph_{\omega+1}, 3,\ldots,3)_{\aleph_0}^2
\]
without assuming the generalised continuum hypothesis.

Main visual structure:
1. A panel titled “The Problem”.
2. A panel titled “The Cardinal Tower” showing:
   \[
   \aleph_0,\aleph_1,\aleph_2,\ldots,\aleph_\omega,\aleph_{\omega+1}.
   \]
3. A central visual:
   - draw \(\aleph_{\omega+1}\) as a large infinite set of points,
   - draw many pair-edges between points,
   - colour or label the edges as countably many colours,
   - show the partition arrow breaking:
     \[
     \aleph_{\omega+1}\not\to(\aleph_{\omega+1},3,\ldots,3)^2_{\aleph_0}.
     \]
4. A notation panel:
   - explain that this is a negative partition relation,
   - it asks for a colouring that avoids the listed homogeneous structures.
5. A GCH panel:
   \[
   \text{Prove it in ZFC, without GCH.}
   \]
6. A context panel:
   Problem of Erdős, Hajnal, and Rado.

Footer:
Status badge: Open
Source: erdosproblems.com/1168

Design rules:
- Keep the house style close to the initial Erdős Atlas posters.
- Use ivory, black, and deep red.
- Keep the text sparse because the notation is already heavy.
- Make the broken arrow \(\not\to\) visually central.
- Make the poster feel abstract, infinite, and set-theoretic.
- Avoid overly colourful graph edges; use subtle edge tones and labels for countably many colours.
