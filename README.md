# Erdős Atlas

A visual gallery of beautiful Erdős problems, conjectures, and mathematical mysteries.

This project creates picturesque posters for open and historically important problems connected to Paul Erdős. Each poster combines:

- the exact problem statement,
- a visual metaphor,
- source notes,
- and a short mathematical explanation.

The goal is not monetization.  
The goal is to build a public mathematical-art archive.

## Gallery

The gallery is published with GitHub Pages.

Each problem has:

```txt
problems/
  <problem-id>-<slug>.md

posters/
